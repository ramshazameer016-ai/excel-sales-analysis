 Excel Sales Analysis Project

 Overview
An end-to-end sales analysis built in Microsoft Excel covering 
data cleaning, formula analysis, pivot tables, charts, and 
a findings report.

 Dataset
- 15 sales records across 3 regions (North, South, East)
- 3 products: Laptop, Phone, Tablet
- Time period: January – May

 Tools Used
- Microsoft Excel
- Formulas: SUM, SUMIF,IF, PROPER, TRIM
- Pivot Tables
- Charts: Column, Pie,Stacked Bar, Line
- Conditional Formatting

Key Findings
1. Total revenue: ₹5,34,000
2. North region leads with ₹2,61,000 in total sales
3. Laptop contributes 37% of total revenue
4. Top performer: Meera with ₹62,000 in sales
5. 2 duplicate rows found and removed during cleaning

 Workbook Structure
- Sheet 1 — Raw Data (original messy dataset)
- Sheet 2 — Cleaned Data (TRIM + PROPER formulas applied)
- Sheet 3 — Analysis (SUMIF, IF, AND)
- Sheet 4- Pivot Analysis
- Sheet 4 — Dashboard (4 charts - Column, Pie, Stacked Bar, Line)
- Sheet 5 — Findings (written recommendations)

 Skills Demonstrated
- Data cleaning (TRIM, PROPER, Remove Duplicates)
- Formula analysis (SUM, SUMIF, IF, AND)
- Pivot Table cross-analysis
- Data visualization (4 chart types)
- Business insight writing
